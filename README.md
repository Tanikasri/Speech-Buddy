# Speech Buddy 🗣️📱

## Overview

Speech Buddy is a picture-to-speech communication app designed to help individuals with speech or cognitive disabilities express themselves easily. It converts selected images into voice output, enabling simple and effective communication.

## Features

* 🖼️ Image-Based Communication: Select images to convey messages
* 🔊 Text-to-Speech: Converts selections into audible speech
* 📚 Custom Categories: Organize images into meaningful groups
* 👶 User-Friendly UI: Simple interface for ease of use
* ⚡ Offline Support: Works without internet for core features

## Tech Stack

**Frontend:** Flutter
**Backend:** (Optional / Future Scope) Node.js
**Database:** Local storage (Hive)
**Speech Engine:** Flutter TTS / Native APIs

## Use Cases

* Assisting non-verbal individuals
* Communication support for children with autism
* Quick expression of daily needs

## Installation

```bash
git clone https://github.com/yourusername/speech-buddy.git
cd speech-buddy
flutter pub get
flutter run
```
